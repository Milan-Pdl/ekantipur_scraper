PHASE 1  (SETUP) 

I was working on Windows so I opened PowerShell and followed the uv docs at
https://docs.astral.sh/uv/ directly:

    powershell -ExecutionPolicy ByPass -c "irm https://astral.sh/uv/install.ps1 | iex"

Then restarted the terminal and ran:

    cd Ekantipur_Scrapper
    uv init
    uv add playwright
    uv run playwright install chromium


PHASE 2 — WEBSITE INSPECTION 

Before writing any prompt I opened both pages in Chrome and used DevTools
(F12 → Ctrl+Shift+C) to manually find every selector I needed.

ENTERTAINMENT PAGE — https://ekantipur.com/entertainment

  Bottom nav HTML I copied:
    <a href="https://ekantipur.com/entertainment"><span>मनोरञ्जन</span></a>
  Selector: bottom-nav a[href*="entertainment"]

  Each article card HTML I copied:
    <div class="category-inner-wrapper">
      <div class="category-description">
        <h2><a href="...">TITLE</a></h2>
        <div class="author-name"><p><a href="...">AUTHOR</a></p></div>
      </div>
      <div class="category-image">
        <figure><img class="loaded" src="CDN_URL" alt="..."></figure>
      </div>
    </div>

  Selectors I wrote down:
    Card wrapper  : .category-inner-wrapper
    Title         : .category-description h2 a
    Author        : .author-name a
    Image         : .category-image img  (data-src first, then src)
    Category      : no badge on this page — default "मनोरञ्जन"

CARTOON PAGE — https://ekantipur.com/cartoon

  Description HTML I copied:
    <div class="cartoon-description">
      <p>गजब छ बा! - अविन</p>
      <div class="date"><p>चैत्र ३, २०८२</p></div>
    </div>

  Image HTML I copied:
    <div class="cartoon-image">
      <figure>
        <a href="FULL_CDN_URL" data-fancybox="gallery">
          <img class="loaded" src="THUMB_CDN_URL">
        </a>
      </figure>
    </div>

  Selectors I wrote down:
    Title + Author : .cartoon-description p:first-child  (split on " - ")
    Image (full)   : .cartoon-image figure a[href]
    Image (thumb)  : .cartoon-image img[src]  (fallback)


PHASE 3 :: PROMPTS  (AI used for code generation only)

PROMPT 1 ::  scrape_entertainment function

  "you are working as a webscrapper in a reputed company. your task is to
  write a function name scrape_entertainment with accepts a parameter page
  (playwright page object) using as per the structure mentioned

  base_url is https://ekantipur.com

  HTML copied:
    <a href='https://ekantipur.com/entertainment'><span>मनोरञ्जन</span></a>
  Selector noted: bottom-nav a[href*='entertainment']

  ENTERTAINMENT PAGE — https://ekantipur.com/entertainment
  Each article card:
    <div class='category-inner-wrapper'>
      <div class='category-description'>
        <h2><a href='...'>TITLE</a></h2>
        <div class='author-name'><p><a href='...'>AUTHOR</a></p></div>
      </div>
      <div class='category-image'>
        <figure><img class='loaded' src='CDN_URL' alt='...'></figure>
      </div>
    </div>

  Selectors noted:
    Card wrapper  : .category-inner-wrapper
    Title         : .category-description h2 a
    Author        : .author-name a
    Image         : .category-image img  (src or data-src for lazy-load)
    Category      : derived from href path segment (e.g. /entertainment/...)"

  Result: AI returned a working scrape_entertainment function.
  I tested it — it returned a valid list of articles. No changes needed.

PROMPT 3 — orchestrator / run_scraper function


  "write a orchestrator function that combines the output coming from
  scrape_entertainment and scrape_cartoon into a output.json file in
  the current directory

  json format:
  {
    'entertainment_news': [ {title, image_url, category, author} ],
    'cartoon_of_the_day': {title, image_url, author}
  }"

  Result: AI returned a run_scraper function that launches a single browser,
  calls both scraper functions, combines the output, and writes output.json.
